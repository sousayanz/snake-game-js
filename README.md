# Snake Game (JavaScript)

Classic Snake game created as my first project using JavaScript.

## Technologies
- HTML5
- CSS3
- JavaScript

## Purpose
This project was created to practice game logic, keyboard input and dynamic updates using JavaScript.

## What I learned
- Game loops
- Keyboard event handling
- Collision detection
- Updating the DOM dynamically

## How to play
- Use the arrow keys to move the snake
- Eat the food to grow
- Avoid hitting the walls and yourself
